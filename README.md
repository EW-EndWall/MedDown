# MedDown
MedDown - Downloader
Youtube video sound downloader.

Features
  - easy linking
  - audio download
  - video download
  - batch download
  - single download
  - select download location
  - bulk delete
  - single delete
  - media name change
  - show total media count
  - show media time
  - show channel name

What to do
  - select quality
  - delete download
  - slow download
